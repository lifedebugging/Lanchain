# Metadata

metdata is generally used for indexing and searching specific topics in vast database.

**Additional uses**:
1. Filtering — retrieve only documents from specific source, date, author, category. You actually did this in your code already.
2. Ranking — prioritize newer documents over older ones using date metadata.
3. Access control — in production, metadata can tag who is allowed to see what.

**In summary**:

metdata is how agents know *where* an answer came from (**the source**). That matters for trust and accuracy.
